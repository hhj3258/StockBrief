# Portfolio Advisor (포터블 스킬)

## 무엇을 하나
보유 종목에 대해 **기계적 익절이 아니라 "테제(논리) 기반"** 으로 **유지 / 추가매수 / 일부매도 / 전량매도** 를 판정한다.
숫자(비중·과열도·시장 국면·별점 가중합·벤치마크 성과)는 **StockBrief 가 결정적으로 계산**하므로, 이 스킬은 그 결과를 *해석*하고 판단·서술·재배치만 한다.

## 입력 (StockBrief 가 제공)
- `Advisor(...).run()` → `BriefingInputs`: `regions`(시장별 국면)·`weights`(비중)·`overheat`·`quotes`(시세·RSI·MA·52주)·`news`·`fx`·`holdings`.
- 또는 `stockbrief.briefing.build_markdown(...)` 마크다운.
- 임계·레지스트리는 `AdvisorConfig`(regions·thresholds: target_profit_pct·stop_loss_pct·max_position_pct·target_cash_pct).

## ⚙️ 결정적 판단 규칙 (먼저 적용)
> 🧮 계산은 StockBrief 가 한다(직접 손으로 계산하지 말 것). 비중·과열도·시장 국면=`metrics.all_regions`/`lib.weights`, 별점 비중적정·가중합=`lib.weight_fit_score`/`lib.star_score`, 벤치마크 성과=`benchmark`, reconcile=`reconcile`, 회고=`retrospect`. 아래는 그 로직의 *해석 명세*.

**(1) 비중 — 분모 고정**: 단일 비중 = 종목평가액 ÷ Σ(보유 평가액). 테마 비중도 같은 분모. (총자산·현금 포함 분모 쓰지 말 것)

**(2) 액션 — 위에서부터 처음 맞는 것 하나** (각 종목 정확히 하나로 확정, 보류·양자택일 금지)
1. 평가액 < 최소 의미 비중 → 별점 ≥ 3.5 & 테제 유효면 **추가매수**(키움), 아니면 **전량매도(정리)**.
2. 손익률 ≤ `stop_loss_pct` → **전량매도(손절)** (명확한 강반등 호재 때만 유지 완화).
3. 설정의 중복 그룹(redundant) → **작은 쪽 전량매도(중복 정리)**. (보완재는 유지)
4. 단일 비중 > `max_position_pct` → **일부매도**(목표=max까지).
5. 그 외 → **유지** (코어 장기지수는 +수익이어도 유지).
- **추가매수**는 "재배치 1순위" *저비중+업황우호* **1종에만**.

**(3) 별점 — 결정적 공식** = `테제0.4 + 밸류추세0.3 + 비중적정0.2 + 상대0.1` (0.5 단위; `lib.star_score`)
- **테제(0~5)**: 개별주=실적/가이던스/목표가여력, ETF=기초지수·테마 추세. ⚠️ 임박 대형이벤트(IPO·실적·정책)는 **양날의 검** — 호·악재 양방향 조사 + "이미 선반영?(뉴스에 팔자)" 점검. 기대만으로 가산 금지.
- **밸류·추세(0~5)**: 정배열&RSI40~70→4.5 · RSI<35+테제유효→4.0 · RSI>70/52주최고근접→3.0 · 역배열→2.0 · (개별주 PER 과도 −0.5).
- **비중적정(0~5)**: `lib.weight_fit_score`(≤10→5·≤20→4·≤25→3.5·≤30→3·>30→1.5, 고변동 −0.5).
- **상대(0~5)**: 재배치1순위→5 · 수급 순매수 지속→4 · 일반→3.5 · 중복/순매도→1.5. **촉매 시간축**: 2주 내 촉매 +0.5 / 2달+ 멀면 −0.5.
> 데이터가 같으면 별점도 같다. 근거 문장만 매일 달라진다(정상). "+20%니 무조건 익절" 금지.

## 시장 국면 해석 (StockBrief `regions`)
- 각 시장(region)이 **독립** 판정됨(오버라이드 없음). 종목은 **자기 region 국면**을 따른다.
- 5단계 톤: 극단공포(적극 분할매수)~극단탐욕(과열 자제). **공포는 방어 아니라 분할매수 기회**(역발상).
- 투자 심리 직접 소스가 없는 시장은 `computed_sentiment`(자체 추정) — 표기 시 "추정" 명시. region 추세 역배열/급락이면 그 시장 "위축(방어)".

## 리밸런싱
- 드리프트 ±5%p(절대) 또는 ±10%(상대) 벗어나면 조정 후보.
- **익절 자금 행선지는 반드시 구체 종목명**(예: "→ DRAM 약 250만원"). 현금비중이 `target_cash_pct`보다 높으면 현금에 쌓지 말고 저비중 우량/코어로 재배치.
- 코어 지수는 "줄이되 비우지 않는다". 포지션 수량은 position-sizer(Fixed Fractional 1% 리스크)로.

## 출력 포맷(예)
```
<이모지> <종목명> | ⭐N.N/5 | 비중 X% | 손익 ±X% | 현재가
- 판단: <유지/추가매수/일부매도/전량매도> — 근거 1~2줄
- 지표: RSI·정/역배열·52주 / (개별주) PER·목표가 / (US) 환율 — 구한 것만
- 실행: N주(≈X) → 재배치 종목명   (행동일 때만)
- 뉴스: [날짜 헤드라인](URL) · … (StockBrief news 전부, 하이퍼링크 필수)
```
시장 구조 통찰(현금·쏠림·수급·이벤트 캘린더)은 '오늘의 고찰' 산문 5~10줄로. **투자 자문 아님 · 정보 제공용.**
